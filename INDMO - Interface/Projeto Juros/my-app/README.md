## App de exemplo
    - Componentes
    - Navigator
    - Screens
    - MaskInput

### Ambiente
Necessário ter instalado:
    - VsCode
    - NodeJS

### Para Executar:
- 1. Clone este repositório
- 2. Abra com VsCode
- 3. Abra o terminal e acess a pasta ./my-app
- 4. Instale as dependências
    ```cms
    yarn
    ```
- 5. Execute o app via web
    ```cmd
    yarn web
    ```
